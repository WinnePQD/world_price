# 🧾 World Price

Aplicativo desenvolvido em Python com Kivy para criação de cartazes de preços em formato A4, com integração em nuvem e geração de PDFs automatizada.

---

## 🚀 Funcionalidades

### 📊 Integração com Firebase

* Busca automática de produtos e preços a partir de planilhas armazenadas no Firebase
* Atualização dinâmica dos dados

### 🖨️ Geração de PDF (A4)

* Criação de cartazes de preços em formato A4
* Layout pronto para impressão

### 👁️ Preview em tempo real

* Visualização do cartaz antes de gerar o PDF
* Ajustes rápidos diretamente na tela

### ✏️ Edição manual

* Alteração de preços manualmente caso necessário
* Correção rápida de inconsistências

### 🆕 Criação do zero

* Criação de cartazes sem depender da planilha
* Total liberdade para o usuário montar o conteúdo

### 📷 Leitura de código de barras

* Captura via câmera do celular
* Identificação rápida de produtos

### 📦 Geração em lote

* Seleção por setor
* Criação automática de múltiplos cartazes
* Ideal para mercados e promoções

### 📤 Compartilhamento

* Envio de PDFs diretamente por:

  * WhatsApp
  * E-mail

---

## 🎯 Objetivo

Facilitar a criação e gestão de cartazes de preços em mercados, reduzindo tempo operacional e erros humanos.

---

## 🛠️ Tecnologias utilizadas

* Python
* Kivy / KivyMD
* Firebase Realtime Database
* Geração de PDF (ReportLab ou similar)
